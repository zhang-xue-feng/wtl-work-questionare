# EO-14117 问卷（Google Forms 自动生成）

本目录提供一个 Google Apps Script，用于自动生成 `EO-14117 合规岗位与平台使用情况调查问卷（内部使用）` 的 Google Form。

## 使用方法（推荐）

1. 打开 `script.google.com` 新建项目（或进入现有项目）。
2. 将 `EO-14117-create-form.gs` 的内容复制到编辑器中（可替换默认的 `Code.gs`）。
3. 保存后，运行函数（任选其一）：
   - `createEO14117FormWizard`：默认版本
   - `createEO14117BigDataEngineerFormWizard`：大数据工程师专用版本（独立出来重新评估影响）
   - `createEO14117OpsDBAFormWizard`：运维/DBA 专用版本（按“隔离权限后无法执行操作/控制”重新评估影响）
4. 首次运行会提示授权（需要允许创建/管理 Google Form）。
5. 运行成功后，在 **执行日志**（Logs）中会输出表单的 **编辑链接**（Edit URL）。

## 生成效果（向导式优化）

- **按需分流**：先问“是否使用（单选）”，若选择“不使用/不确定”，自动跳到下一个平台，避免填写无意义的后续题。
- **逐平台采集依赖程度**：仅当选择“使用”，继续填写该平台的“使用目的（多选）”、“不可访问影响（单选）”与“可选备注”。（因此不再需要单独的“平台使用目的汇总”）
- **大数据工程师专用版本差异**：选择“不确定”也会继续进入“使用目的/不可访问影响”，用于更严格的影响评估（避免不确定导致缺失）。
- **运维/DBA 专用版本差异**：选择“不确定”也会继续进入影响评估，并且影响选项更贴近“隔离权限后无法执行操作/控制”（避免低估为“影响效率/无影响”）。
- **Part 2 末尾支持补充**：在 Part 2 的最后新增“其他平台（可选）”，可补充 1 个未覆盖平台及其使用目的（可选）。
- **每个平台页面结构（仅选择“使用”时）**：
  - 第 1 页：是否使用（单选）
  - 第 2 页：使用目的（多选，仅使用时填写） + 不可访问影响（单选） + 可选备注

> 如你希望改成“每个平台一页（不分页）”或“按分类分页（每类一页）”，我也可以把脚本再给你一版。
